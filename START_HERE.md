## 🎯 What Each BMI Gets

### 🔵 Underweight (BMI < 18.5)
```
Goal: BUILD MUSCLE
├─ Eat: +400 kcal/day surplus
├─ Protein: 1.9g/kg (max muscle gain)
├─ Train: 4-5x/week strength
├─ Eat: Calorie-dense foods (oats, nuts, milk)
└─ Timeline: Gain 0.5-1kg monthly
```

### 🟢 Normal Weight (BMI 18.5–24.9) ✨ NEW
```
Goal: BODY COMPOSITION (lean muscle, low fat)
├─ Eat: TDEE or TDEE+200 (maintenance/lean gains)
├─ Protein: 1.6g/kg (optimal for recomposition)
├─ Train: 4x strength + 1-2x cardio
├─ Focus: Compound lifts, balanced macros
├─ Sleep: 7-9 hours (critical for muscle growth)
└─ Timeline: 2-6 months for visible change
```

### 🟡 Overweight (BMI 25–29.9)
```
Goal: LOSE FAT (preserve muscle)
├─ Eat: -400 kcal/day deficit
├─ Protein: 1.4g/kg (high for preservation)
├─ Train: 3-4x strength + 2-3x cardio
├─ Focus: Low-impact cardio, cardio 20-30 min
└─ Timeline: 0.5kg/week loss = 10-15 weeks to normal
```

### 🔴 Obese (BMI ≥ 30)
```
Goal: URGENT FAT LOSS (with health monitoring)
├─ Eat: -500 kcal/day aggressive deficit
├─ Protein: 1.3g/kg (critical preservation)
├─ Train: 2-3x light strength + daily low-impact cardio
├─ Alert: Medical check-up recommended
├─ Focus: Joint protection, daily walking/swimming
└─ Timeline: 1kg/week = 12-18 months to normal BMI
```

## ❓ FAQs

**Q: What's new in this update?**
A: All 4 BMI categories now have recommendations. **Normal Weight (18.5–24.9) is NEW!** ✨

**Q: Why different protein targets?**
A: Each goal needs different protein: Underweight needs more (1.9) for muscle gain, Normal needs moderate (1.6) for balance, Overweight/Obese need high (1.4/1.3) for preservation during deficit.

**Q: Can I customize the recommendations?**
A: Yes, modify the `calcBMI()` function in either HTML file (lines shown in documentation).

**Q: Is this mobile-friendly?**
A: Yes, fully responsive (tested at 375px, 768px, 1200px).

**Q: Does it work offline?**
A: Yes, localStorage allows offline use (multi-user version).

**Q: Can users print the guidance?**
A: Yes, print `BMI_QUICK_REFERENCE.md` for a guide.
