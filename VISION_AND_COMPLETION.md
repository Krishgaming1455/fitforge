## 🎓 Key Learning Points

### The Protein Logic:
```
Underweight (1.9g/kg): Building phase, surplus allows "wasting" some on fat
Normal (1.6g/kg): Balanced, sufficient for muscle without excess
Overweight (1.4g/kg): High for preservation during deficit
Obese (1.3g/kg): Conservative preservation during aggressive deficit
```

### The Calorie Logic:
```
Underweight: +400 = ~2,800 cal deficit for 1lb muscle (sustainable)
Normal: TDEE or +200 = recomposition with minimal fat gain
Overweight: -400 = ~3,500 cal deficit weekly for 1lb fat loss
Obese: -500 = ~3,500 cal deficit weekly for 1kg fat loss
```

### The Training Logic:
```
Underweight: 4-5x strength (aggressive growth)
Normal: 4x strength + 1-2x cardio (balanced)
Overweight: 3-4x strength + 2-3x cardio (preserve + burn)
Obese: 2-3x light + daily cardio (joint safety + calorie burn)
```
